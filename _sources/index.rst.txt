     <h1>PySAL</h1>
     <p>PySAL, the Python spatial analysis library, is an open source
         cross-platform library for geospatial data science with an emphasis on
         geospatial vector data written in Python. It supports the development of
         high level applications for spatial analysis. It consists of the following subpackages which could be
         individually installed or installed as a meta package
         to obtain the full functionality of PySAL:</p>
