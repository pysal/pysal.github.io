    PySAL

    PySAL, the Python spatial analysis library, is an open source cross-platform library for geospatial data science with an emphasis on geospatial vector data written in Python. It supports the development of high level applications for spatial analysis. It consists of the following subpackages which could be individually installed or installed as a meta package to obtain the full functionality of PySAL:

    Lib

    LibPySAL
    Core of PySAL containing computational geometry, graph construction, and read/write tools

    Explore

    Esda
    Statistics and inference for exploratory spatial data analysis
    Giddy
    Statistics and inference for GeospatIal Distribution Dynamics
    Pointpats
    Planar point pattern analysis library
    Inequality
    Spatial inequality indices and inference
    Region
    Spatially-explicit regionalization algorithms
    Spaghetti
